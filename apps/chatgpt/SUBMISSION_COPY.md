## Full description

Negative Lab Converter helps users convert a single image inside ChatGPT without forcing them into a multi-step desktop workflow. The app opens a conversion workspace, analyzes the source format, recommends practical export settings, and lets the user preview the source and result before downloading the converted file.

The app is designed for straightforward utility work such as:

- turning HEIC photos into lighter JPEG files for blogs or uploads
- exporting an image for web or social use with smaller dimensions
- comparing source and converted output before download
- checking whether a format such as CRAW, CR3, TIFF, or HEIC is supported

Format support should be described conservatively. RAW support is best-effort, and `CR3` / `CRAW` should be described as preview-first rather than guaranteed full-fidelity decode.
