# Negative Lab Converter Launch Checklist

## 목표

이 문서는 현재 프로젝트 상태를 기준으로 실제 출시 준비에 필요한 작업을 정리한다.

우선순위는 다음 순서로 잡는다.

1. Windows/mac 직접 판매 시작
2. Microsoft Store 등록
3. Android 출시
4. iOS 출시

## 현재 상태

- Windows 설치본 생성 가능
- macOS DMG 생성 가능
- Android/iOS 셸 프로젝트 존재
- RAW / HEIC / TIFF / 일반 이미지 변환 동작
- 진단 패널 포함

## 지금 바로 막히는 항목

- Windows 코드 서명 없음
- macOS 서명 및 notarization 없음
- 개인정보처리방침 없음
- 지원/문의 페이지 없음
- 결제 시스템 없음
- 실제 출시용 스크린샷/설명문 없음
- RAW 회귀 테스트 묶음 없음

## 1차 출시안: 직접 판매

가장 빠른 출시 방식은 Windows 설치본과 macOS DMG를 직접 판매하는 것이다.

### 필수 작업

- [ ] 제품명 최종 확정
- [ ] 공식 웹사이트 또는 판매 페이지 만들기
- [ ] 개인정보처리방침 작성
- [ ] 환불 정책 작성
- [ ] 지원 이메일 만들기
- [ ] 앱 소개 문구 작성
- [ ] 스크린샷 6장 이상 준비
- [ ] Windows 코드 서명 인증서 구매 및 적용
- [ ] macOS Developer 계정 등록
- [ ] macOS 서명 및 notarization 적용
- [ ] 버전 업데이트 규칙 정리

### 권장 판매 경로

- Gumroad
- Lemon Squeezy
- Paddle
- 자체 웹사이트 + Stripe

## 가격안

현재 앱 특성상 구독보다 1회 구매 또는 평생 라이선스가 더 자연스럽다.

### 옵션 A

- 기본판 무료
- Pro 업그레이드 19~29달러

### 옵션 B

- 유료 단일 앱 14.99~24.99달러

### 옵션 C

- 무료 체험판
- 정식판 24.99달러

## Pro 차별화 후보

돈을 받을 기능은 기본 변환 외에 명확한 차이가 있어야 한다.

- 배치 변환
- 폴더 단위 처리
- 메타데이터 유지 옵션
- 카메라별 프리셋
- 대용량 파일 최적화
- 자동 파일명 규칙
- 변환 이력 저장
- 워터마크 / 출력 프리셋

## 직접 판매 기준 출시 최소선

아래가 되면 판매 시작 가능으로 본다.

- [ ] Windows 서명된 설치본
- [ ] macOS notarized DMG
- [ ] 지원 페이지
- [ ] 개인정보처리방침
- [ ] 구매 페이지
- [ ] 20개 이상 실파일 회귀 테스트
- [ ] 치명적 변환 실패 0건

## 권장 운영 문구

이 앱은 사진을 서버에 업로드하지 않고, 기기 안에서 직접 변환을 처리한다.

이 문장은 제품 소개, 랜딩 페이지, 개인정보처리방침에 반복적으로 쓰는 것이 좋다.

## 권장 출시 순서

1. Windows 직접 판매
2. macOS 직접 판매
3. Microsoft Store 등록
4. Android
5. iOS

## 메모

모바일 스토어에서 앱 내부 디지털 판매를 붙이면 스토어 결제 규칙을 따라야 한다.
직접 판매는 데스크톱이 훨씬 단순하다.
