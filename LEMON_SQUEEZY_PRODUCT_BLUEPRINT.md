# Lemon Squeezy Product Blueprint

최종 업데이트: 2026-03-30

## 권장 상품

- Product name: `Negative Lab Converter`
- Product type: 디지털 다운로드
- Billing model: 1회 구매
- Price: `$24`

## 상품 한 줄 설명

RAW, HEIC, TIFF 사진을 서버 업로드 없이 로컬에서 바로 변환하는 데스크톱 유틸리티

## 상품 설명 본문

Negative Lab Converter는 사진을 외부 서버에 올리지 않고, 기기 안에서 바로 변환하는 오프라인 사진 유틸리티입니다.

다음 작업에 적합합니다.

- HEIC를 JPEG / PNG로 정리
- TIFF, PNG, WebP 포맷 변환
- RAW 계열 빠른 미리보기와 변환
- 가로/세로 크기 조정
- 품질 조절

포함 항목:

- Windows 설치본
- macOS DMG
- 1.x 업데이트

## 권장 버튼 문구

- 메인 버튼: `데스크톱 정식판 구매`
- 보조 버튼: `웹 데모 먼저 보기`
- 체크아웃 버튼 앞 카피: `지금 바로 로컬 사진 변환기 정식판을 시작합니다.`

## 권장 전달 방식

### 가장 단순한 v1

- 결제 완료 후 thank-you 페이지에서 다운로드 링크 제공
- 결제 완료 이메일에도 Windows/mac 다운로드 링크 제공
- 버전 업데이트는 수동 재다운로드 방식으로 시작

### v1.5

- 구매 후 라이선스 키 메일 발송
- 앱 첫 실행 시 라이선스 입력

## 권장 환불 문구

- `구매 후 7일 이내, 기술적 문제로 핵심 기능을 사용할 수 없는 경우 환불을 검토합니다.`

## 권장 지원 문구

- `영업일 기준 2일 이내 답변`
- `문제 파일 형식, OS 버전, 오류 화면을 함께 보내 주세요.`
