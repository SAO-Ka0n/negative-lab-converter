# Negative Lab Converter Store Copy Draft

## 한 줄 소개

RAW, HEIC, TIFF, PNG, JPEG 이미지를 기기 안에서 바로 변환하는 오프라인 사진 변환 앱

## 짧은 설명

사진을 서버에 올리지 않고 이 기기 안에서 바로 변환합니다. 검증된 사진 형식을 불러오고 PNG, JPEG, WebP, TIFF로 저장할 수 있습니다.

## 긴 설명

Negative Lab Converter는 사진을 업로드하지 않고 로컬에서 직접 변환하는 오프라인 사진 변환 앱입니다.

다음 작업에 적합합니다.

- 카메라 RAW 파일 빠른 미리보기
- HEIC / HEIF를 JPEG 또는 PNG로 변환
- TIFF, PNG, WebP 포맷 정리
- 가로/세로 크기 조정
- JPEG 품질과 배경색 제어

핵심 특징

- 오프라인 변환
- 검증된 사진 형식 위주 지원
- 빠른 비교 보기
- Windows / macOS 웹직판 + Android / iOS 유료 스토어 앱 구조
- 진단 패널 포함

지원 형식 표기 원칙

- 실제 판매 문구에는 회귀 코퍼스로 검증된 형식만 남깁니다.
- 미검증 카메라 조합이나 실험적 경로는 FAQ 또는 릴리스 노트로 내립니다.

지원 형식

- 입력: PNG, JPEG, WebP, AVIF, HEIC, HEIF, TIFF, RAW 계열
- 출력: PNG, JPEG, WebP, TIFF

적합한 사용자

- 사진 작업자
- 카메라 사용자
- 로컬 보안이 중요한 사용자
- 파일 포맷을 빠르게 정리해야 하는 사용자

## 출시용 짧은 문구 후보

- 사진을 서버에 올리지 않는 오프라인 변환기
- RAW와 HEIC를 빠르게 정리하는 로컬 사진 유틸리티
- 기기 안에서 바로 끝내는 사진 포맷 변환 앱
