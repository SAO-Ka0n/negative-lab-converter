# 개인정보처리방침

최종 업데이트: 2026-03-26

Negative Lab Converter는 사용자의 사진을 가능한 한 기기 안에서 직접 처리하도록 설계된 오프라인 사진 변환 앱입니다.

## 1. 수집하는 정보

앱 자체는 기본적으로 다음 정보를 외부 서버로 전송하지 않습니다.

- 사용자가 선택한 이미지 파일
- 변환된 결과 파일
- 파일 내용

기본 앱 동작만으로는 사진 파일이 개발자 서버로 업로드되지 않습니다.

## 2. 파일 처리 방식

사용자가 선택한 사진은 사용자의 기기 안에서 직접 읽고 변환합니다.

앱은 다음 작업을 로컬에서 수행할 수 있습니다.

- 이미지 미리보기 생성
- 포맷 변환
- 리사이즈
- 품질 조정

## 3. 진단 정보

앱에는 실행 환경과 디코더 상태를 확인하기 위한 진단 패널이 포함될 수 있습니다.

진단 패널의 정보는 기본적으로 사용자의 기기 안에서만 표시됩니다.
사용자가 직접 복사하거나 공유하지 않는 한 자동 전송되지 않습니다.

## 4. 제3자 서비스

현재 기본 앱은 사용자 사진을 외부 서버에 업로드하지 않습니다.

다만 다음과 같은 외부 서비스가 추후 추가될 수 있습니다.

- 결제 서비스
- 웹사이트 분석 도구
- 고객 지원 도구
- 업데이트 확인 서비스

이 경우 실제 사용되는 서비스와 처리 목적을 별도로 고지합니다.

## 5. 결제

앱이 유료로 판매되거나 인앱 결제를 제공하는 경우, 결제 처리는 스토어 또는 결제 제공업체를 통해 이루어질 수 있습니다.
개발자는 결제 제공업체가 처리하는 카드 정보 전체에 직접 접근하지 않습니다.

## 6. 데이터 보관

앱이 로컬에서 처리한 사진 파일은 사용자의 기기 저장소와 사용자가 선택한 저장 위치에만 존재합니다.

개발자가 별도로 수집하거나 보관하지 않습니다.

## 7. 아동의 개인정보

이 앱은 아동을 대상으로 설계되지 않았습니다.

## 8. 문의

개인정보 처리와 관련한 문의는 공식 판매 페이지 또는 앱 내 제품 / 지원 정보 패널에 표시되는 최신 지원 채널을 사용합니다.

## 9. 변경

이 개인정보처리방침은 서비스 변경, 기능 추가, 법적 요구사항에 따라 업데이트될 수 있습니다.
